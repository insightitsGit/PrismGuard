# Support

## Community (open source)

- **Bugs & features:** [GitHub Issues](https://github.com/insightitsGit/PrismGuard/issues)
- **Docs:** [`docs/`](docs/) — start with [integration-guide.md](docs/integration-guide.md)
- **Install help:** run `prismguard doctor` and `prismguard eval self-check` after install

Before opening an issue, please check:

1. You installed a recent release: `pip install -U "prismguard[guard-model]"`
2. ONNX weights are present: `prismguard-model download`
3. CI is green on `main`: [Actions](https://github.com/insightitsGit/PrismGuard/actions)
